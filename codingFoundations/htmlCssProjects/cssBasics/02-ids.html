CSS IDs

An ID is a UNIQUE identifer

ID Notes

  • In CSS, an ID starts with an octothrop (pound, hashtag, #)
  • Example uses for IDs: Targeting a specific header or about section
  • Each ID should only be used ONCE per page - they are unique!
    This is important with javascript later on
  • Use for one off styling

IDs vs. Classes

IDs

This ID will be linked in the CSS page by a #

You cannot use two IDs on one element. If you do, no style for any of the IDs will apply to that element

Classes

Elements can have any number of classes

For example, this element has two classes

You CAN use an ID twice, but you shouldn't.
