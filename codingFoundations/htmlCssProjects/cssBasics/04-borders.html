Borders

The border property allows you to specify the style, width, and color of an element's border.

Border Style

The border-style property can take up to four values (top right bottom left). Here are some examples of what is allowed:

  • Dotted
  • Solid
  • Double
  • Dashed
  • Etc

Border Width

Note: border-width will not work unless you have a border-style set

Border width can be set to a specific size (px, em, etc) or you can use one of the following presets: thin, medium, thick. Border-width property can take up to four values (top right bottom left)

Example border widths

Border Colors

Note: border-color will not work unless you have a border-style set.

Border-color can be set in the following ways:

  • Color name
  • Hex code
  • RGB
  • Transparent
